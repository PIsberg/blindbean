# AGENTS.md

Instructions for AI coding agents working in this repository.

This file is intentionally a pointer. The authoritative, maintained guidance
lives in the two platform files below — read the one for your platform, and do
not duplicate their content here (it would drift).

| File | Audience |
|:-----|:---------|
| [CLAUDE.md](CLAUDE.md) | Claude Code / Claude agents |
| [GEMINI.md](GEMINI.md) | Gemini agents |

Both files carry the same project guardrails, generated by
[VibeTags](https://github.com/PIsberg/vibetags) from the `@AI*` annotations in
the source. The guardrail sections are **generated** — change the annotations
in the Java source, not the markdown between the VibeTags markers. The root files carry the merged
block for every module; each annotated module also owns a `.claude/rules/` of
role-grouped topic files (scoped by its `.vibetags-roles`), so working inside a
module surfaces just that module's guardrails.

Whichever agent you are: read your platform's file before making changes.
